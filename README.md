# Bread-Cancer-Support-System
Proyek ini bertujuan untuk mengembangkan solusi Machine Learning yang terintegrasi penuh untuk klasifikasi sel Kanker Payudara. Kami memanfaatkan Dataset Breast Cancer Wisconsin (Diagnostic) dari Kaggle sebagai dasar pelatihan.

Fokus utamanya adalah membangun model prediksi yang andal—berdasarkan fitur morfologi inti sel hasil pemeriksaan Fine Needle Aspiration (FNA)—kemudian mengemasnya dalam aplikasi Streamlit yang memiliki dua peran krusial:
- Akselerasi Keputusan Klinis: Menyediakan alat validasi cepat bagi tenaga medis, melengkapi analisis laboratorium dengan prediksi berbasis AI.
- Pemberdayaan Pasien: Menawarkan platform edukasi yang mudah diakses dan informatif, meningkatkan kesadaran publik terhadap Kanker Payudara.
